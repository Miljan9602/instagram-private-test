search()

Hashtag.php : 137

Search for hashtags.

public search( $query : string [, $excludeList : string[]|int[] = [] ] [, $rankToken : string|null = null ] ) : SearchTagResponse

Gives you search results ordered by best matches first.

Note that you can get more than one "page" of hashtag search results by excluding the numerical IDs of all tags from a previous search query.

Also note that the excludes must be done via Instagram's internal, numerical IDs for the tags, which you can get from this search-response.

Lastly, be aware that they will never exclude any tag that perfectly matches your search query, even if you provide its exact ID too.

Parameters
$query : string

Finds hashtags containing this string.

$excludeList : string[]|int[] = []

Array of numerical hashtag IDs (ie "17841562498105353") to exclude from the response, allowing you to skip tags from a previous call to get more results.

$rankToken : string|null = null

(When paginating) The rank token from the previous page's response.

markStoryMediaSeen()

Mark TagsStoryResponse story media items as seen.

public markStoryMediaSeen( $hashtagFeed : TagsStoryResponse , $items : \InstagramAPI\Response\Model\Item[] [, $skippedItems : \InstagramAPI\Response\Model\Item[] = [] ] ) : MediaSeenResponse

The "story" property of a TagsStoryResponse only gives you a list of story media. It doesn't actually mark any stories as "seen", so the user doesn't know that you've seen their story. Actually marking the story as "seen" is done via this endpoint instead. The official app calls this endpoint periodically (with 1 or more items at a time) while watching a story.

This tells the user that you've seen their story, and also helps Instagram know that it shouldn't give you those seen stories again if you request the same hashtag feed multiple times.

Tip: You can pass in the whole "getItems()" array from the hashtag's "story" property, to easily mark all of the TagFeedResponse's story media items as seen.

Parameters
$hashtagFeed : TagsStoryResponse

The hashtag feed response object which the story media items came from. The story items MUST belong to it.

$items : \InstagramAPI\Response\Model\Item[]

Array of one or more story media Items.

$skippedItems : \InstagramAPI\Response\Model\Item[] = []

Array of one or more story media Items.

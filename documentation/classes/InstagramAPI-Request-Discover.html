Search for Instagram users, hashtags and places via Facebook's algorithm.

public search( $query : string [, $latitude : string = null ] [, $longitude : string = null ] [, $excludeList : array = [] ] [, $rankToken : string|null = null ] ) : FBSearchResponse

This performs a combined search for "top results" in all 3 areas at once.

Parameters
$query : string

The username/full name, hashtag or location to search for.

$latitude : string = null

(optional) Latitude.

$longitude : string = null

(optional) Longitude.

$excludeList : array = []

Array of grouped numerical entity IDs (ie "users" => ["4021088339"]) to exclude from the response, allowing you to skip entities from a previous call to get more results. The following entities are supported: "users", "places", "tags".

$rankToken : string|null = null

(When paginating) The rank token from the previous page's response.
